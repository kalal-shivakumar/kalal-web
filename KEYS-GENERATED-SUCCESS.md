### 2. Add Public Key to GitHub
1. Copy your public key from `public-key.asc`
2. Go to GitHub → Settings → SSH and GPG keys
3. Click "New GPG key"
4. Paste the entire key (including `-----BEGIN...-----END-----`)
5. Click "Add GPG key"

### 3. Create Signed Commits
```powershell
cd your-repo
git add .
git commit -S -m "Your commit message"  # -S signs the commit
git push origin main
```

### 4. Verify Signed Commits on GitHub
- Your commits will show a "Verified" badge on GitHub
- Anyone can verify the signature using your public key

### 5. Use in GitHub Actions
```yaml
- name: Import GPG key
  run: |
    echo "${{ secrets.GPG_PRIVATE_KEY }}" | gpg --import
    git config --global user.signingkey "${{ secrets.GPG_KEY_ID }}"
    git config --global commit.gpgsign true
```
